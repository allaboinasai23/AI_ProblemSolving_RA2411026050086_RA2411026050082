📘 How It Works

Sudoku is modelled as a CSP — each empty cell is a variable with domain {1–9}.

Constraints:

• Each row has digits 1–9 without repetition
• Each column has digits 1–9 without repetition
• Each 3×3 subgrid has digits 1–9 without repetition

The solver uses Backtracking with the MRV (Minimum Remaining Values) heuristic — it picks the cell with the fewest legal options first, making the search much faster.
